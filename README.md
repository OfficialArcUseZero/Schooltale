# Schooltale
DSA Project - Salud Group, PLM BSIT 1-2
--------------------------------
This is an indie made adventure game called "Schooltale." 
The game's story include a new transfer student Kirito walking by his new school, Tomori High.
Along the way, he sees a wide range of characters that make his school life interesting.

---------------------------------

Gameplay Details:
Kirito must avoid and dodge obstacles in order to progress the story. 
Failure to do so and losing three lives results in a game over screen.

---------------------------------

Version Details:
Current commit "498728e" is a demo version with one main adversary, Ryuga the Bully! 
Avoid his backpacks in order to claim victory.
Further commits will add to the story in incremental steps.
